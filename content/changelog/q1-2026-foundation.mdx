# Q1 2026 — Foundation Release

The foundation of NXπ is now generally available.

## What shipped

**MCP Integration Fabric** — NXπ is a first-class MCP host. Connect SAP HANA Cloud, S/4HANA, Salesforce, and any MCP-compatible system through the open protocol. Native support for SSE and WebSocket transports.

**Hybrid RAG** — Vector search + BM25 with Reciprocal Rank Fusion. Default retrieval strategy achieves 91%+ recall@10. Source attribution on every answer.

**Agent Builder** — Create single agents and multi-agent pipelines. Token accounting per agent per run. Full call tree in the audit log.

**Zero-Trust Governance** — Per-user RBAC, label-based policies, MCP server scoping, model allowlists, PII detection and masking, append-only audit log.

**Self-hosted deployment** — Run NXπ entirely within your infrastructure. PostgreSQL + pgvector + Ollama. Nothing leaves your perimeter.
