# Q2 2026 — SAP + Salesforce Connectors

## What shipped

**SAP HANA Cloud MCP connector** — Native integration with SAP HANA Cloud via MCP. Real-time data access, no replication. Joule-compatible.

**Salesforce MCP connector** — Agentforce-native integration. Access Opportunities, Accounts, Cases, and custom objects through MCP. Bi-directional tool support.

**Visual Workflow Builder** — DAG-based workflow automation with 9 node types: Input, Output, LLM, Tool, Condition, HTTP, Template, Agent, Note. Cron, webhook, and API triggers.

**Finance-specific reasoning flows** — Pre-built workflows for: margin analysis, variance analysis, financial reconciliation, working capital insights, and scenario modeling against SAP data.
